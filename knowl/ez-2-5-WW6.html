Exercise 6. Chain rule with function values.

Given \(F(2)=1, F'(2)=1, F(4)=3, F'(4)=4\) and \(G(3)=7, G'(3)=2, G(4)=2, G'(4)=7\text{,}\) find each of the following. (Enter dne for any derivative that cannot be computed from this information alone.)

  1. \(H(4)\) if \(H(x)=F(G(x))\)

  2. \(H'(4)\) if \(H(x)=F(G(x))\)

  3. \(H(4)\) if \(H(x)=G(F(x))\)

  4. \(H'(4)\) if \(H(x)=G(F(x))\)

  5. \(H'(4)\) if \(H(x)=F(x)/G(x)\)
