Exercise 6. Chain rule with function values.

Given \(F(3)=1, F'(3)=3, F(4)=5, F'(4)=3\) and \(G(1)=4, G'(1)=7, G(3)=4, G'(3)=7\text{,}\) find each of the following. (Enter dne for any derivative that cannot be computed from this information alone.)

  1. \(H(3)\) if \(H(x)=F(G(x))\)

  2. \(H'(3)\) if \(H(x)=F(G(x))\)

  3. \(H(3)\) if \(H(x)=G(F(x))\)

  4. \(H'(3)\) if \(H(x)=G(F(x))\)

  5. \(H'(3)\) if \(H(x)=F(x)/G(x)\)
