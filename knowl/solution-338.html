Solution 5.5.5.3.1.

\begin{equation*} \frac{7 x + 2}{x^2-3x+2} = \frac{7 x+2}{(x-1)(x-2)} = \frac{A}{x-1} + \frac{B}{x-2}. \end{equation*}
Multiplying by \((x-1)\) and \((x-2)\text{,}\) this becomes
\begin{equation*} 7 x+2 = A(x-2)+B(x-1), \end{equation*}
or
\begin{equation*} A+B = 7 \quad\hbox{and}\quad -2A - B = 2. \end{equation*}
Solving this system yields \(A = -9\) and \(B = 16\text{.}\) Then
\begin{equation*} \int\frac{7 x+2}{x^2-3x+2}dx = \int\left(\frac{16}{x-2} - \frac{9}{x-1}\right)\,dx = 16\ln\!\left(\left|x-2\right|\right)-9\ln\!\left(\left|x-1\right|\right)+C. \end{equation*}
