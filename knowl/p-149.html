Using the same calculation as in (a), it follows that a 170 lb person swimming uses 9.8 calories per minute. Thus a 170 lb person swimming for one hour uses a total of \(( 9.8 )(60) = 588\) calories. On the other hand, a 220 lb person bicycling uses 11.9 calories per minute. Thus a 220 lb person bicycling for a half-hour uses a total of \(( 11.9 )(30) = 357\) calories. By comparing the values 588 and 357 we obtain the answer.
