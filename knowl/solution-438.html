Solution 7.4.3.3.1.

\begin{equation*} \frac{dy}{y^2} = (4 + t)\,dt, \end{equation*}
so
\begin{equation*} \int \frac{dy}{y^2} = \int (4 + t)\,dt, \end{equation*}
and
\begin{equation*} -\frac{1}{y} = {1\over2}(4 + t)^2 + C, \end{equation*}
giving
\begin{equation*} y=-\frac{2}{(4 + t)^2 + A}, \end{equation*}
where \(A = 2C\text{.}\) Since \(y=6\) when \(t=1\text{,}\) we have
\begin{equation*} 6 = -\frac{2}{5^2 + A},\quad \hbox{ so } \quad 6 A + 150 = -2, \end{equation*}
and \(A = -{{76\over3}}\text{.}\) Thus
\begin{equation*} y = -\frac{2}{(4 + t)^2 - {{76\over3}}}. \end{equation*}
