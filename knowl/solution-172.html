Solution 2.7.3.3.1.

\(x\text{.}\)
\begin{equation*} \frac{1}{\sqrt{1 - x^{4} y^2}} (2 x y + x^{2} \frac{dy}{dx}) = y^{2} + 2 x y \frac{dy}{dx}, \end{equation*}
so, clearing the denominator,
\begin{equation*} 2 x y + x^{2} \frac{dy}{dx} = \sqrt{1 - x^{4} y^2} (y^{2} + 2 x y \frac{dy}{dx}). \end{equation*}
Thus
\begin{equation*} ( x^{2} - (2 x y) \sqrt{1 - x^{4} y^2} ) \frac{dy}{dx} = y^{2} \sqrt{1 - x^{4} y^2} - 2 x y, \end{equation*}
and finally
\begin{equation*} \frac{dy}{dx} = \frac{y^{2} \sqrt{1 - x^{4} y^2} - 2 x y}{x^{2} - (2 x y) \sqrt{1 - x^{4} y^2}}. \end{equation*}
