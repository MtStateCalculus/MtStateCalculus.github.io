Solution 6.5.5.2.1.

\begin{equation*} \int_{2}^{\infty} 5xe^{-x^{2}} dx = \lim_{b \rightarrow \infty} \int_{2}^b 5xe^{-x^{2}} dx = \left.\lim_{b \rightarrow \infty} \left({{-5} \over 2} e^{-x^2}\right) \right|_0^b = \end{equation*}
\begin{equation*} \lim_{b \rightarrow \infty} \left({{-5} \over 2} e^{-b^2} - {{-5} \over 2} e^{-2^2}\right) = 0 + {5\over 2} e^{-4} = {5\over 2} e^{-4}. \end{equation*}
This integral converges to \({5\over 2} e^{-4}\text{.}\)
