Solution 5.3.5.6.1.

\(w = -\cos\!\left(q\right)\text{,}\) so that \(dw = \sin\!\left(q\right)\,dq\text{.}\) Then \(w(\pi) = 1\) and \(w(3\pi/2) = 0\text{,}\) so that
\begin{equation*} \int_{\pi}^{3\pi/2}\, e^{-\cos\!\left(q\right)}\cdot \sin\!\left(q\right) \,dq = \int_1^0\,e^w\,dw = e^w\bigg|_1^0 = 1 - e. \end{equation*}
