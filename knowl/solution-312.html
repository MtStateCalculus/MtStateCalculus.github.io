Solution 5.3.5.4.1.

\(w = e^{4 x} + 7\text{.}\) Then \(dw = 4\,e^{4 x}\,dx\text{,}\) so that
\begin{equation*} \int\,\frac{e^{4 x}}{7 + e^{4 x}}\,dx = {1\over 4}\int\,{1\over w}\,dw = {1\over 4}\,\ln(|w|) + C = {1\over 4}\,\ln(|e^{4 x} + 7|) + C. \end{equation*}
Because \(e^{4 x} + 7\) is always positive, we can drop the absolute values from the argument of the natural log.
