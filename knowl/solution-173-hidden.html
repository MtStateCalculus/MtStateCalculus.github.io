so, clearing the denominator,
\begin{equation*} 3 x^{2} y + x^{3} \frac{dy}{dx} = \sqrt{1 - x^{6} y^2} (y^{3} + 3 x y^{2} \frac{dy}{dx}). \end{equation*}
Thus
\begin{equation*} ( x^{3} - (3 x y^{2}) \sqrt{1 - x^{6} y^2} ) \frac{dy}{dx} = y^{3} \sqrt{1 - x^{6} y^2} - 3 x^{2} y, \end{equation*}
and finally
\begin{equation*} \frac{dy}{dx} = \frac{y^{3} \sqrt{1 - x^{6} y^2} - 3 x^{2} y}{x^{3} - (3 x y^{2}) \sqrt{1 - x^{6} y^2}}. \end{equation*}
