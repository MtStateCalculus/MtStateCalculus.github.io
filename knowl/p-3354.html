We find inflection points by determining where \(f''(x)\) changes sign. \(f'(x) = 2\cdot 4x^{3}+19\cdot 3x^{2}-15\cdot 2x\text{,}\) so \(f''(x) = 2\cdot 4\cdot 3x^{2}+19\cdot 3\cdot 2x-30 = 6(4 x - 1)(x + 5)\text{.}\) So we see that \(f''(x)\) changes sign at \(x = 1/4\) and \(x = -5\text{.}\)
