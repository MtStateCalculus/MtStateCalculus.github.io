\(\displaystyle c_n = \frac{f^{(n)}(0)}{n!}\text{,}\) e.g. \(\displaystyle c_0 = \frac{f(0)}{0!} = 0\text{.}\) \(\displaystyle c_1 = \frac{f'(0)}{1!}\text{;}\) \(\displaystyle f'(x)=4 \arctan(5 x) + 4 x\frac{1}{(5 x)^2+1}5 = 4 \arctan(5 x) + \frac{20 x}{25 x^2+1}\text{;}\) \(c_1 = 0\text{.}\) \(\displaystyle c_2 = \frac{f''(0)}{2!}\text{;}\) \(\displaystyle f''(x) = \left( 4 \arctan(5 x) + \frac{20 x}{25 x^2+1} \right)'\)\(\displaystyle = \frac{20}{25 x^2+1} + \frac{20(25 x^2+1)-20 x25\cdot 2x}{(25 x^2+1)^2}\)\(\displaystyle = \frac{500 x^2 + 20}{(25 x^2+1)^2} + \frac{20 - 500 x^2}{(25 x^2+1)^2} = \frac{40}{(25 x^2+1)^2}\text{;}\) \(\displaystyle c_2 = \frac{40}{2} = 20\text{;}\) and so on. Another way is to use the power series for arctan: \(\displaystyle \arctan(t) = \sum_{n=0}^{\infty} (-1)^n \frac{t^{2n+1}}{2n+1} = t-\frac{t^3}{3}+\frac{t^5}{5}-\frac{t^7}{7}+...\text{.}\) Then \(\displaystyle 4 x \arctan(5 x) = 4 x \sum_{n=0}^\infty (-1)^n \frac{(5 x)^{2n+1}}{2n+1} = \sum_{n=0}^\infty (-1)^n \frac{4\cdot 5^{2n+1}}{2n+1} x^{2n+2}\)\(\displaystyle = 20 x^2 - \frac{500}{3} x^4 + \frac{12500}{5} x^6 - \frac{312500}{7} x^7+...\text{.}\) So \(c_0=0\text{,}\) \(c_1=0\text{,}\) \(c_2=20\text{,}\) \(c_3=0\text{,}\) \(c_4=-\frac{500}{3}=-166.666666666667\text{...}\) This is an alternating series. It converges if and only if \(\displaystyle \lim_{n\to\infty}\frac{4 \cdot (5 x)^{2n+1}}{2n+1}=0\text{.}\) This limit is equal to 0 if and only if \(|5 x| \le 1\text{,}\) so \(\displaystyle |x|\le \frac{1}{5}\text{.}\) Therefore the radius of convergence is \(\displaystyle \frac{1}{5}\text{.}\)
