Solution 5.6.6.1.1.

(a)\(\int_{0}^{3}\,e^x\,dx = e^x\bigg|_{0}^{3} = e^3 - e^0 \approx 19.0855\ldots\)(b) Computing the sums directly, since \(\Delta x=1.5\text{,}\) we have LEFT(2) \(= 1.5\cdot e^0 + 1.5\cdot e^{1.5} \approx 8.2225\) RIGHT(2) \(= 1.5\cdot e^{1.5} + 1.5\cdot e^{3} \approx 36.8509\) TRAP(2) \(= \frac 1 2 ( 8.2225 + 36.8509 ) \approx 22.5367\) MID(2) \(= 1.5\cdot e^{0.75} + 1.5\cdot e^{3\cdot 0.75} \approx 17.4072\) and SIMP(2) = (2 MID(2) + TRAP(2))/3 \(\approx 19.117\text{.}\) And the errors are just the actual value minus the approximate values, or error(LEFT(2)) = 10.863; error(RIGHT(2)) = -17.7654; error(TRAP(2)) = -3.4512; error(MID(2)) = 1.6784; and error(SIMP(2)) = -0.0315. (c) Similarly, since \(\Delta x = 0.375\text{,}\) we have LEFT(4) = 12.8149; error = 6.2707 RIGHT(4) = 27.129; error = -8.0435 TRAP(4) = 19.9719; error = -0.8864 MID(4) = 18.6455; error = 0.44 SIMP(4) = 19.0876; error = -0.0021. (d) For LEFT and RIGHT, we expect the error to go down by 1/2, and this is very roughly what we see. For MID and TRAP, we expect the error to go down by 1/4, and this is approximately what we see. For SIMP, we expect the error to go down by 1/16, again as observed.
