We find inflection points by determining where \(f''(x)\) changes sign. \(f'(x) = 6\cdot 4x^{3}+34\cdot 3x^{2}-18\cdot 2x\text{,}\) so \(f''(x) = 6\cdot 4\cdot 3x^{2}+34\cdot 3\cdot 2x-36 = 12(6 x - 1)(x + 3)\text{.}\) So we see that \(f''(x)\) changes sign at \(x = 1/6\) and \(x = -3\text{.}\)
